# 自动元胞机
- 可以模拟传染病的传播, 火灾的蔓延, 新闻的传播, 核裂变

- 一些想法
    - 文中在SIS模型中提到 `Transimission Rate`(转化率)在22%和23%之间有巨大的差异, 就是说病毒的传染性并不是连续的影响. 22%及以下演变到最后都趋近于消失, 23%及以上最后会蔓延到所有地方. 所以我们尽管看到疫情有一丝好转, 也还不能松懈, 不能输在那1%上
